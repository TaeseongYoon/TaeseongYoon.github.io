# Taesung Yoon — Personal Academic Website

This is a static GitHub Pages website.

## How to use

1. Create a GitHub repository named:

```text
YOUR_GITHUB_ID.github.io
```

2. Upload these files to the repository root:

```text
index.html
style.css
```

3. Replace placeholders in `index.html`:

- `your.email@kaist.ac.kr`
- `YOUR_GITHUB_ID`
- `YOUR_ID` for Google Scholar
- `#` links for paper/code URLs
- `files/cv.pdf` if you want to upload your CV

4. Commit and push.

Your website will be available at:

```text
https://YOUR_GITHUB_ID.github.io
```

## Optional

Create a `files/` folder and upload your CV as:

```text
files/cv.pdf
```

You can also replace the circular `TY` placeholder with a profile photo by editing the `.photo-placeholder` block in `index.html`.
